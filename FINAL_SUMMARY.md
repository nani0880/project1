## 🎨 Design System

### **Typography Hierarchy:**
```
Headlines: Playfair Display
- H1: 4rem (64px), weight 900
- H2: 3rem (48px), weight 800
- H3: 2rem (32px), weight 700
- Letter spacing: -0.02em

Body Text: Inter
- Regular: 18px, weight 400
- Light: 18px, weight 300
- Line height: 1.8
- Letter spacing: 0.02em

UI Elements: Inter
- Buttons: weight 600
- Navigation: weight 500
- Captions: weight 300, uppercase
```

### **Animation System:**
```
Letter-by-Letter:
- Duration: 0.6s
- Easing: cubic-bezier(0.34, 1.56, 0.64, 1)
- Stagger: 0.03-0.05s
- Transform: translateY + rotateX

Word-by-Word:
- Duration: 0.5s
- Easing: cubic-bezier(0.22, 1, 0.36, 1)
- Stagger: 0.1s
- Transform: translateY

Fade-Up:
- Duration: 0.8s
- Easing: cubic-bezier(0.22, 1, 0.36, 1)
- Transform: translateY(60px) to 0
```
